# Clothy - AI穿搭闺蜜 🌸

> OpenClaw 生态首个生活场景 Skill

你的专属 AI 穿搭顾问，像闺蜜一样懂你的风格。无论是日常穿搭、衣橱管理还是逛街种草，一句话搞定。支持**多人衣橱**，全家人的穿搭都能管！

## ✨ 核心能力

### 1. 👨‍👩‍👧‍👦 多人衣橱管理（NEW）
**一个 Skill 管理全家人的穿搭：**
- 👩 妈妈（主用户）→ 管理自己 + 帮家人搭配
- 👨 老公 → 独立画像 + 独立衣橱
- 👧 小孩 → 儿童专属字段 + 特殊建议
- 👴 老人 → 独立画像 + 独立衣橱

**智能识别：**
- "今天我穿什么？" → 当前成员推荐
- "今天老公穿什么？" → 自动识别 + 切换推荐
- "给小孩录入一件夹克" → 自动识别 + 录入到小孩衣橱

### 2. 📸 衣橱智能管理
**拍照录入** — 拍下衣服自动识别类型、颜色、季节  
**自动分类** — 上衣/下装/外套/鞋子智能归档  
**视觉识别** — 支持多模态模型识别，降级时本地分析，最差也能手动录入

### 3. 🌤️ 每日穿搭推荐
**场景感知** — "今天约会穿什么？""明天有雨怎么搭？"  
**风格匹配** — 根据你的用户画像（身材/风格/偏好）智能推荐  
**成套搭配** — 上衣+下装+鞋子+配饰一键生成

### 4. 🛒 逛街种草咨询
**实时参谋** — 看中一件衣服？拍照发给 Clothy  
**搭配建议** — "这件和我衣橱里的米色风衣能搭吗？"  
**购买决策** — 基于现有衣橱，给出买或不买的建议

### 5. 📊 衣橱分析洞察
**配置分析** — 了解你的衣橱构成（颜色分布、季节占比、风格偏向）  
**购物指南** — "缺一件黑色基础款打底衫""夏季单品太多，建议补充秋季"  
**理性消费** — 避免重复购买，优化衣橱性价比

### 6. 🏪 关注店铺追踪（NEW）
**关注店铺** — 收藏喜欢的淘宝店铺、抖音博主
**自动追踪** — 定期检查上新，AI分析推荐
**智能提醒** — 新品适合你？主动通知

## 🧒 儿童专属功能

**儿童特殊处理：**
- 出生日期管理 → 根据年龄给建议
- 校服信息 → 学校着装要求
- 喜欢的卡通形象 → 艾莎、小猪佩奇、奥特曼

**智能穿搭建议：**
- 3岁以下 → 方便换尿布、面料柔软
- 3-6岁 → 方便活动、避免小装饰
- 天气热 → 透气面料（孩子好动易出汗）
- 下雨天 → 准备雨衣（孩子喜欢踩水）

## 🏗️ 技术亮点

### 架构设计
- **多人数据隔离** — 每个成员独立衣橱、独立画像
- **智能成员识别** — 从文本自动识别目标成员
- **会话状态管理** — 切换成员后保持上下文
- **OpenClaw 原生集成** — 语义理解为主，关键词辅助路由

### 用户画像系统
- **身材数据** — 身高、体重、三围，自动计算身材类型
- **风格偏好** — 喜欢的/避免的风格、颜色
- **儿童画像** — 年龄、学校、喜欢的卡通、课外活动

### 智能降级策略
- 多模态识别 → 本地分析 → 手动输入
- 三级优雅降级，无 Vision 也能用
