# 变更日志

## [0.5.0] - 2026-03-25

### 新增
- 🏠 **多人衣橱管理** — 支持家庭成员独立衣橱
  - 成员添加/删除/切换
  - 智能成员识别
  - 儿童专属字段（年龄、身高体重）
- 📸 **AI 服装识别** — 多模态模型识别服装属性
  - 品类、颜色、风格自动识别
  - 支持无 API 时的本地降级
- 🛒 **淘宝导入** — 从订单/购物车/收藏夹导入
- 🏪 **关注店铺** — 追踪店铺新品
- 👗 **搭配推荐** — 基于天气、场景、风格推荐

### 改进
- 重构主路由架构
- 优化数据库结构
- 完善消息格式

### 技术债务
- 命令解析器意图识别优化中
